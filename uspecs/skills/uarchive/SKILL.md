# uarchive

Parse user input as `[options]`.

Identify the root directory of the plugin (PLUGIN_ROOT) and run `bash {PLUGIN_ROOT}/bin/softeng.sh action uarchive [options]` and follow the instructions in the output.

Options: `--change-folder <path>`, `--all`
